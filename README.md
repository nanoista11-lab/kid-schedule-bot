# 🤖 בוט לוז ילדים — טלגרם

בוט שמשלח תזכורות אוטומטיות לטלגרם עם קישור לשליחה לוואצאפ.

---

## שלב 1 — פתיחת בוט בטלגרם (2 דקות)

1. פתח/י טלגרם וחפש/י `@BotFather`
2. שלח/י `/newbot`
3. בחר/י שם לבוט (למשל: `לוז המשפחה`)
4. בחר/י שם משתמש (חייב להסתיים ב-bot, למשל: `family_schedule_bot`)
5. קבל/י **Token** — שמור/י אותו (נראה כך: `7234567890:AAFxxxxxx`)

---

## שלב 2 — קבלת Chat ID שלך

1. שלח/י הודעה לבוט שיצרת (חפש/י אותו בטלגרם)
2. כתוב/י `/start`
3. הבוט ישלח לך את ה-**Chat ID** שלך — שמור/י אותו

---

## שלב 3 — העלאה ל-Railway (חינמי)

1. כנס/י לאתר: https://railway.app
2. התחבר/י עם GitHub (צור/י חשבון אם אין)
3. צור/י repo ב-GitHub עם כל הקבצים מהתיקייה הזו
4. ב-Railway לחץ/י **"New Project"** → **"Deploy from GitHub repo"**
5. בחר/י את ה-repo שיצרת
6. לחץ/י על הפרויקט → **Variables** → הוסף/י:
   - `TELEGRAM_BOT_TOKEN` = הטוקן מ-BotFather
   - `TELEGRAM_CHAT_ID` = ה-Chat ID שקיבלת
7. Railway יבנה ויעלה אוטומטית — לוקח ~2 דקות

---

## שלב 4 — הוספת אירועים

פתח/י את `index.html` בדפדפן (פשוט גרור/י את הקובץ).
מלא/י אירוע → לחץ/י הוסף → יורד קובץ `events.json`.
העתק/י את הקובץ לתיקיית `data/` בשרת ב-Railway.

**דרך Railway:**
1. לחץ/י על הפרויקט
2. **Files** → `data/events.json`
3. הדבק/י את התוכן של הקובץ שהורדת

---

## פקודות הבוט

| פקודה | תיאור |
|-------|-------|
| `/start` | פתיחה + קבלת Chat ID |

| `/tomorrow` | אירועי מחר |
